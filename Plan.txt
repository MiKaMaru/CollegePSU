#Lesson 0
    На занятиях по Основам программирования, курс 3, мы будем изучать язык программирования C# и некоторые его особенности.
    Для работы нам потребуется поставить среду разработки для работы с этим языком программирования.
    Мы будем использовать Visual Studio (2015, 2017, 2019), необходима поддержка классических приложений .NET и языка программирования C#.
    Необязательно, но рекомендую поставить English версию VS 2019, для того чтобы облегчить себе поиск по возникающим проблемам во время разработки.
